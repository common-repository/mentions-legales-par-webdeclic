Génère un shortcode pour les mentions légales qui sont obligatoires sur les sites internet en France.

== Description ==
Ce plugin permet d'afficher les mentions légales obligatoires, via l'utilisation d'un shortcode.
L'administrateur n'a qu'à inscrire les informations du site, et de copier le shortcode sur une page dénommée par exemple "Mentions légales".

= Fonctionnalités =
 * Génération des mentions légales pour les sites particuliers
 * Génération des mentions légales pour les sites professionnels

= Usage =
 1. Aller dans Mentions légales
 2. Remplir le formulaire avec les informations adéquates
 3. Copier le shortcode [wbd_mentions_legales], et coller dans une nouvelle page WordPress dénommée "Mentions légales"

== Installation ==
1. Aller dans *Plugins* et cliquer sur *Ajouter*
2. Chercher *Mentions légales par Webdeclic*
3. Cliquer sur *Installer* sur la fiche du plugin
4. Activer le plugin

== FAQ ==

=  Qu'est-ce que les mentions légales obligatoires ? =
Les mentions légales obligatoires sont des informations requises par la loi française et doivent être présentes sur tous les sites internet. Elles fournissent des renseignements sur l'identité du propriétaire du site, les conditions d'utilisation, la protection des données, et d'autres éléments juridiques importants.

=  Ce plugin est-il compatible avec les sites professionnels et particuliers ? =
Oui, ce plugin permet de générer les mentions légales aussi bien pour les sites professionnels que pour les sites particuliers. Il suffit de remplir le formulaire avec les informations appropriées et le shortcode généré affichera les mentions légales correspondantes.

=  Comment utiliser ce plugin ? =
Pour utiliser ce plugin, suivez les étapes suivantes :
Accédez à la section "Mentions légales" dans le menu WordPress.
Remplissez le formulaire avec les informations adéquates.
Copiez le shortcode [wbd_mentions_legales] généré.
Collez ce shortcode dans une nouvelle page WordPress intitulée "Mentions légales".

Les captures d'écran du plugin sont disponibles dans la section "Screenshots" du README. Elles vous donneront un aperçu visuel de l'interface d'administration ainsi que de l'affichage du shortcode sur le site.
